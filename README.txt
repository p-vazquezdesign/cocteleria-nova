#Proyecto de Desarrollo Web - Coctelería Nova 

## Descripción del Proyecto 
Este proyecto consiste en el desarrollo de una página web semática construida con HTML5 y CSS3. Presenta una interfaz de usuario monimalista y limpia, priorizando la jeraquía tipográfica, la accesibilidad y la legibilidad. 

##Tecnologías Utilizandas 
-HTML5: Estructuración semática (<header>, <nav>, <main>, <section>, <aside>, <footer>)
-CSS3: Maquetación y estilos visuales
-Git & Github: Control de versiones 

##Estructura del Proyecto 
/
|-index.html 
|-style.css 
|-README.txt

## Ventajas de HTML (Notas Académicas)
-Semántica Mejorada: El uso de etiquetas semáticas reemplaza la excesiva dependencia de etiquetas div. 
-Accesibilidad: Permite a los lectores de pantalla navegar por la estructura lógica del documento. 
-Optimización SEO: Los motores de búsqueda identifican claramente la jerarquía del contenido.
